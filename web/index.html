Vos sources

Tout ce que vous suivez retrouve une forme simple.

Flux rassemble votre web personnel dans une seule fenêtre claire, pensée pour lire avant de réagir.

  • Ajoutez un site ou un flux, Flux détecte souvent la bonne source pour vous.
  • Retrouvez RSS, Atom, blogs, newsletters et chaînes YouTube au même endroit.
  • Organisez vos dossiers, vos favoris et vos habitudes de lecture sans vous disperser.

Lecture calme

Lire sans timeline, sans agitation, sans détour.

Le contenu reste central. Les sources, le contexte et le partage restent proches, mais ne prennent jamais le dessus.

  • Une lecture concentrée qui laisse la place à l’article.
  • Une vue immersive quand vous voulez aller au fond d’un sujet.
  • Un vrai rythme de lecture, pas un flux qui vous pousse.

Newsletter

Transformer ses flux en une synthèse calme et utile.

Flux peut composer une newsletter à partir de vos sources pour reprendre le fil sans rouvrir vingt onglets.

  • Une vue récapitulative faite depuis vos propres abonnements.
  • Des résumés locaux pour aller plus vite quand vous le souhaitez.
  • Une expérience pensée pour Mac, jusque dans l’audio et les notifications.
