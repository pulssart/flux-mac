Flux for Safari

Enable the extension

Open Safari Settings, then turn on Flux Safari Extension.

Flux Safari Extension is on. You can now use it from Safari’s Extensions button.

Flux Safari Extension is off. Open Safari Settings to turn it on.

1Open Safari Settings.
2Go to Extensions.
3Enable Flux Safari Extension.

If the extension is missing, enable the Develop menu in Safari Advanced settings, then use Develop > Allow Unsigned Extensions.
